# Phase 13 — GEO & SEO Upgrades

> The all-in plan to recover from Google's May-2026 Core Update, finally attack off-site
> **authority** (our #1 blocker), and become the most-cited AI-tools reference across ChatGPT,
> Perplexity, Gemini, and Claude. Runs in parallel with a conversion fix.

## 60-second picture
We have **world-class technical SEO and the best freshness automation in the category, but near-zero
off-site authority** — so 94% of our Google appearances are on page 3+, impressions are *falling*
post-core-update, ~700 pages won't index, no LLM cites us, and only 5 people signed up last month.
Phase 13 fixes the demotion (make pages genuinely original, not template-filled), builds authority
with verified operator-approved automations, engineers the cross-source "consensus signal" that makes
AIs cite us, and fixes the funnel.
